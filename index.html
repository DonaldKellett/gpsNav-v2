gpsNav - Version 2 by Donald Leung

Welcome to the second version of my gpsNav app!

Since Version 1 (the original gpsNav app), there have been a few updates:

  1. Fixed glitches in Version 1 so the init() function can take any number (and any combination) of settings
  2. The new version allows the user to set the default image and text (message) displayed whenever there is an error - this is done by passing the properties image and text directly inside the error object.
  3. Added a new setting: getPositionOnce - setting this to true disables GPS tracking - the visitor's location is only used once

For more info, please read the README.md file in this repository.

P.S. If you haven't checked out the HTML code of this demo page, be sure to throw this into a text editor - feel free to mess with the gpsNav initialization code (there will be instructions in the comments within the <script> tags)!
